Title- Resource-Aware and Personalized Federated Learning via Clustering Analysis

Abstract

Federated Learning (FL) has become an emerging paradigm for collaborative learning in large-scale distributed systems with a massive number of networked smart edge devices, such as smartphones, wearables, connected vehicles, or sensors. Compared to other distributed learning approaches, FL allows edge devices to train models without sharing raw data, which achieves privacy-preserving machine learning in real application scenarios. This brings great opportunities for deploying AI approaches in future intelligent systems. However, the iterative nature of FL does not eliminate the network congestion problem. Indeed, the communication overhead at each iteration becomes a challenge for complex models, large-scale applications, and high-frequency updates. Moreover, In Wireless Networks (WN), an energy-efficient transmission is a fundamental challenge since the energy resource of edge devices is restricted. In addition, for Human Activity Recognition (HAR) scenarios, the existing action recognition system mainly focuses on a unified model, i.e., it does not provide users with personalized recognition of activities in smart applications such as personalized health care. Finally, an effective evaluation of each client's data behavior to the federated global model when aggregating local models becomes an essential challenge in FL applications. Therefore, this thesis aims to develop and implement optimized FL algorithms to address the abovementioned challenges using clustering optimization.
